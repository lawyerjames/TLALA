# 臺南市地政法學會靜態網站

這是一個可部署到 GitHub Pages 的靜態官方網站雛形，內容整理自公開資料與使用者提供的 Facebook 粉專網址。

## 檔案結構

```text
index.html
assets/
  css/styles.css
  images/hero-land-law.png
  images/TLALA_LOGO.jpg
  js/main.js
```

## GitHub Pages 部署

1. 將本資料夾內容推送到 `lawyerjames/TLALA`。
2. 到 GitHub repo 的 `Settings` → `Pages`。
3. Source 選擇 `Deploy from a branch`。
4. Branch 選擇 `main` 或 `master`，資料夾選 `/root`。

## 待補資料

- Facebook 粉專照片、活動相簿與貼文可再整理成「最新消息」或「活動花絮」頁面。
- 若學會有正式 logo、章程、入會辦法或活動報名表，可加入 `assets/` 或新增獨立頁面。
- 聯絡資訊與理事長資訊以臺南市政府社會局人民團體名冊為準，日後可依正式會務資料更新。
